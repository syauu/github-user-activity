**GitHub Activity Viewer**

A simple Python CLI tool that fetches and displays a user's recent GitHub public activity using GitHub REST API.
This project is my first attempt working with API.

**Requirements**
- Python 3.11
- requests library

**API Endpoint Used**
https://api.github.com/users/{username}/events
